# How to run the project locally: 🏃🏽‍♂️

### Run with Docker(recommended) 🐋: ⬇️

```docker-compose up```

or

### Download the dependencies: ⬇️

```npm i```
<br>

```yarn```

#### Run app: 🏎️

```npm run dev```
<br>

```yarn dev```

#### Run with specific port: 🚪

```npm run dev -- --port [your-port]```
<br>

```yarn dev -- --port [your-port]```
